RakutenAdvertisingAds iOS SDK

Rakuten Advertising Ads SDK provide the ability to show ads.

Requirements

  • iOS 10.0+
  • Xcode 11+
  • Swift 5+

Integration

Use CocoaPods to install RakutenAdvertisingAds pod. If you dont have Cocoapods installed follow this guide for intallations. If you have Cocopods already installed add the following lines in your Podfile:

source 'https://github.com/CocoaPods/Specs.git'
source 'https://github.com/Rakuten-Advertising-Developers/Specs.git'

target 'YOUR_TARGET_NAME' do

  use_frameworks!
  pod 'RakutenAdvertisingAds'

end

Run the following command from your project’s Podfile location

pod install --repo-update

Setup

Import required in all places where you would like to use SDK

import RakutenAdvertisingAds
Targeting

Setting information related to GDPR

RakutenAdvertisingAds.shared.targeting.subjectToGDPR = true
RakutenAdvertisingAds.shared.targeting.gdprMasking = false
RakutenAdvertisingAds.shared.targeting.gdprConsentString = <Some Consent String>
Advertising Identifier

Provide user’s advertising identifier (import AdSupport required)

RakutenAdvertisingAds.shared.adSupport.isTrackingEnabled = ASIdentifierManager.shared().isAdvertisingTrackingEnabled
RakutenAdvertisingAds.shared.adSupport.advertisingIdentifier = ASIdentifierManager.shared().advertisingIdentifier.uuidString
Geo data

Provide user’s location info (import CoreLocation required)

class AppDelegate: UIResponder, UIApplicationDelegate {

    private lazy var locationManager: CLLocationManager = {
        let manager = CLLocationManager()
        manager.delegate = self
        return manager
    }()

    func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
        ...
        configureLocation()
        ...
        return true
    }

    func configureLocation() {

        let status = CLLocationManager.authorizationStatus()
        switch status {
        case .notDetermined:
            locationManager.requestWhenInUseAuthorization()
        case .denied, .restricted:
            print("Location Services are disabled")
        case .authorizedAlways, .authorizedWhenInUse:
            locationManager.startUpdatingLocation()
        @unknown default:
            break
        }
    }
}

extension AppDelegate: CLLocationManagerDelegate {

    func locationManager(_ manager: CLLocationManager, didUpdateLocations locations: [CLLocation]) {

        guard let location = locations.last else {
            return
        }
        RakutenAdvertisingAds.shared.geoInfo.location = location
    }
}
Logging

For debugging purpose you can enable network logs. RakutenAdvertisingAds.shared.logger property is responsible for this process.

RakutenAdvertisingAds.shared.logger.enabled = true

Ad Formats

Create an ad unit first. You will need that ad unit’s ID later to request ads from RSSP server.

Banner Ads

In your view controller declare a BannerAdsConfig instance and container view, where ads should be placed.

class ViewController: UIViewController {
    ...
    var bannerAdsConfig: BannerAdsConfig?
    var placeholderView: UIView = UIView()
    ...
}

placeholderView should have valid autolayout constraints and added to view hierarchy

  • Instantiate your bannerAdsConfig, passing Ad Unit. This is typically done in -viewDidLoad.
  • Register your view controller as the bannerAdsConfig’s delegate. (optional step)
  • If you know the size of the banner you’d like to request, pass it to adUnitSize property.
  • Provide refresh time interval or set to .never if you would like a single load. (optional step)
  • Load an ad
override func viewDidLoad() {
    super.viewDidLoad()

    self.bannerAdsConfig = BannerAdsConfig(adUnit: <YOUR_AD_UNIT>, containerView: placeholderView)
    self.bannerAdsConfig?.delegate = self
    self.bannerAdsConfig?.adUnitSize = .mediumRectangle
    self.bannerAdsConfig?.requestTimeInterval = .seconds(10)
    self.bannerAdsConfig?.startLoading()
}

Optionally declare that your view controller implements the BannerAdsConfigDelegate protocol.

extension ViewController: BannerAdsConfigDelegate {

    func didLoadData(bannerAdsConfig: BannerAdsConfig) {
        //your implementation
    }

    func didFailLoadData(bannerAdsConfig: BannerAdsConfig, error: Error) {
        //your implementation
    }

    func willLeaveApplication(from bannerAdView: BannerAdsConfig) {
        //your implementation
    }
}

Native Ads

In your view controller declare a NativeAdsConfig instance and container view, where native ads should be placed.

class ViewController: UIViewController {
    ...
    var nativeAdRequest: NativeAdsConfig?
    var placeholderView: UIView = UIView()
    ...
}

placeholderView should have valid autolayout constraints and added to view hierarchy

  • Instantiate adView as UIView subclass instance that confirms to NativeAdRanderableView protocol.
  • Instantiate your nativeAdRequest, passing Ad Unit value. This is typically done in -viewDidLoad.
  • Register your view controller as the nativeAdRequest’s delegate. (optional step)
  • Pass the list of expected assets.
  • Provide refresh time interval or set to .never if you would like single load. (optional step)
  • Load an ad
override func viewDidLoad() {
    super.viewDidLoad()

    let adView = NativeAdView.instance

    self.nativeAdRequest = NativeAdsConfig(adUnit: <YOUR_AD_UNIT>,
                                           containerView: placeholderView,
                                           nativeView: adView)
    self.nativeAdRequest?.delegate = self
    self.nativeAdRequest?.assets = NativeAdAsset.allCases
    self.nativeAdRequest?.requestTimeInterval = .seconds(10)
    self.nativeAdRequest?.startLoading()
}

Optionally declare that your view controller implements the NativeAdsConfigDelegate protocol.

extension ViewController: NativeAdsConfigDelegate {

    func didLoadData(nativeAd: NativeAdRepresentable, nativeAdsConfig: NativeAdsConfig) {
        //your implementation
    }

    func didFailLoadData(nativeAdsConfig: NativeAdsConfig, error: Error) {
        //your implementation
    }

    func willLeaveApplication(from nativeAdsConfig: NativeAdsConfig) {
        //your implementation   
    }
}
